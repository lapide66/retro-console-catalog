# Mobile

Camada inicial do app mobile em Expo.

Objetivo atual:
- validar a estrutura `apps/mobile`
- consumir a camada compartilhada em `shared/`
- exibir uma lista simples de consoles por geração

Comandos a partir da raiz:

```bash
npm run dev:mobile
npm run android:mobile
npm run ios:mobile
```

Observação:
- as dependências do app mobile ainda precisam ser instaladas localmente no próximo `npm install`
