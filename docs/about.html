Sobre

Retro Console Catalog

Catálogo técnico de consoles por geração, organizado como índice editorial e alimentado por dados públicos.

O projeto reúne 25 consoles em 9 gerações, da primeira geração doméstica (1972) até a oitava geração (2017). Cada ficha inclui especificações técnicas, preço de lançamento, vendas totais, dimensões, conectividade e uma curiosidade sobre a história do console no Brasil.

Todos os dados têm fontes verificáveis. As imagens são fotos reais dos aparelhos obtidas do Wikimedia Commons, sob licenças de domínio público ou uso livre.

O projeto é estático — sem backend, sem dependências externas, sem build. Funciona diretamente no navegador e está pronto para publicação no GitHub Pages.

Tecnologia

HTML · CSS · JavaScript puro · JSON como fonte de dados

Aviso

Projeto independente, sem afiliação com nenhum fabricante. Dados de vendas e preços são históricos e podem variar conforme a fonte.
