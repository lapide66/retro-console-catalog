Arquivo editorial de videogames

Contexto, critérios e navegação do catálogo em um só lugar.

O Retro Console Catalog organiza consoles por geração e apresenta cada aparelho com resumo histórico, ficha técnica, indicadores de mercado, dados físicos e uma curiosidade sobre sua presença no Brasil.

A proposta é manter a leveza de um site estático enquanto melhora clareza, hierarquia e presença visual. Tudo continua funcionando apenas com HTML, CSS, JavaScript e JSON.

Leitura rápida

  • Use a home para navegar direto entre os consoles por geração.
  • Abra a ficha individual para ver mercado, hardware e contexto no Brasil.
  • Abra referências para validar preço, vendas e origem das imagens.

Índice principal

A página inicial agora foi simplificada para priorizar usabilidade. Ela concentra somente o catálogo e reduz distrações antes da exploração dos consoles.

Escopo

25 consoles Organizados em 9 gerações históricas.
100% estático Sem backend, sem build obrigatório, sem dependências extras.

Tecnologia

  • HTML5 para estrutura e navegação entre páginas.
  • CSS para identidade visual, responsividade e hierarquia.
  • JavaScript puro para renderizar conteúdo a partir do JSON.
  • JSON como fonte central de dados do catálogo.

Conteúdo

  • Especificações técnicas como CPU, GPU, RAM, mídia e resolução.
  • Preço de lançamento, vendas totais e links de referência.
  • Dimensões, peso, conectividade e retrocompatibilidade quando disponível.
  • Curiosidade sobre o contexto brasileiro de cada console.

Aviso

Projeto independente, sem afiliação com fabricantes. Os dados de preço e vendas são históricos e podem variar conforme a fonte consultada.
